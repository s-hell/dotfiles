# bash config

I have create a [folder](configs/.bash.d/) with all necessary file. Just put this folder in your homedirectory and source [~/.bash.d/bashdreader](configs/.bash.d/bashdreader) in your .bashrc


# sway config
I'm using sway with swaybar on all my machines. The configs are nearly identical. [This](configs/.config/sway) is the config i use on my laptop.

Be aware: If you qnt to use it, you have to modify pathes etc. There are also configs for my [touchpad](configs/.config/sway/touchpad) and my [keyboard](configs/.config/sway/keyboard).
